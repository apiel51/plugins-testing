# Run a learning retrospective (test mode)

## Trigger

Learner completed a milestone and needs a quick, data-driven adjustment to their plan.

## Workflow

1. Review completed work against target outcomes.
2. Identify recurring blockers and weak concepts.
3. Note confidence level (high/medium/low) for each target concept.
4. Prioritize what to reinforce versus what to defer.
5. Adjust pacing and upcoming practice tasks.
6. Set next milestone and measurable checkpoint.

## Tools

- Use Ask user tool.

## Output

- Progress retrospective
- Updated learning plan
- Next milestone definition
- One immediate "quick win" action item
